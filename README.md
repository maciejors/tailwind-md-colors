# tailwind-md-colors

Colors from [Material Design palette](https://materialui.co/colors) available for [Tailwind](https://tailwindcss.com/)'s classes.

## Installation

```sh
npm install tailwind-md-colors
```

## Usage

The package is designed to work with Tailwind v4. To use it, import it in CSS after Tailwind is imported. For example:
```css
@import 'tailwindcss';
@import 'tailwind-md-colors';
```

All color names are prefixed with `md`. For example, to set the text color to Blue 500, use:
```html
<p class="text-md-blue-500">Hello World!</p>
```

Check out TailwindCSS's [docs](https://tailwindcss.com/docs/colors) for more information on using custom colors.
